# 家庭邮轮假期 · Claude Code 插件

为家庭旅客设计的邮轮产品，支持按目的地和活动筛选，确保每位家庭成员都能享受旅程。

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/craftwave-skill-9-claude-plugin
/plugin install craftwave-skill-9@craftwave-skill-9-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **家庭邮轮假期**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
